# chill.exe

chill.exe is a Discord bot made for fun and casual use with friends. It includes a mix of utility, entertainment, and a simple card-based economy system.

## Features

- Basic economy system with balance and daily rewards  
- Card collection system with inventory management  
- Chat moderation tools  
- Simple entertainment commands  

## Commands

### General
- `!helpme` — shows the list of available commands  
- `!profile` — displays your profile and stats  

### Entertainment
- `!joke` — sends a random joke (can be inappropriate)  
- `!sound` — plays a secret sound in your current voice channel  

### Economy
- `!balance` — shows your current balance  
- `!daily` — claim your daily reward  

### Cards
- `!inv` — shows your card inventory  
- `!sell <number>` — sells a card by its number  
- `!sell all` — sells all cards except legendary ones  

### Moderation
- `!clear <amount>` — deletes a specified number of messages (up to 500)  

## Notes

- Some commands may require permissions  
- The bot is still in development, so features may change  
- The economy and card system are being expanded over time  

## About

This project was created as a practice project for learning Python and Discord bot development.
